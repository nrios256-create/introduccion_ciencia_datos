# Introducción a Ciencia de Datos

Este repositorio contiene materiales, notebooks y archivos utilizados en la materia **Introducción a Ciencia de Datos**.

## Contenido

Dentro de este repositorio se encuentra una carpeta con:

- Notebooks en Python (colab)
- Archivos CSV y Excel para prácticas
- Material de clase
- Ejercicios de análisis de datos

## Objetivo

El objetivo de este repositorio es organizar los archivos trabajados en clase y facilitar el acceso a los recursos utilizados en el curso de Ciencia de Datos.

## Herramientas utilizadas

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Archivos CSV / Excel
